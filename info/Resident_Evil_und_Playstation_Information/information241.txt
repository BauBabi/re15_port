Sony Computer Entertainment Inc. CONFIDENTI
AL
For textured models, it is necessary to modify the batch file or the make file.
In either case, a temporary file (textures.tim) must be created that links all the
necessary texture files. A mark ('0') is added to the end of the linked file so that
the end can be detected when it is loaded.
Change the batch file or the make file as shown below
For batch files (where a.tim, b.tim, c.tim are the textures used)
@REM You need ONE concatenated texture file with an end mark ('0').
copy /b a.tim+b.tim+c.tim textures.tim
echo 0>> textures.tim
pqbload textures.tim 80500000
For make files (where a.tim, b.tim, c.tim are the textures used)
TEXTURES = a.tim+b.tim+c.tim
TIMFILE = textures.tim
...
