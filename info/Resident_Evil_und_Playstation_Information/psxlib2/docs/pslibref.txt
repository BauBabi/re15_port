PS Library Reference
~~~~~~~~~~~~~~~~~~~~

When in doubt, read the header files in the include directory (read
them anyway :).

NB: For SPU functions, see spulib2.txt and spulib2.html.



GPU Functions:
~~~~~~~~~~~~~~

1. Initialising the GPU

int         GPU_PALorNTSC(void);

returns 1 if PAL, else 0 if NTSC

void        GPU_SetPAL(PsxInt32 palFlag);

Use palFlag = MODE_PAL or 0 (for NTSC)


PsxInt32    GPU_Reset(PsxInt32 mode);

mode = ?


PsxInt32    GPU_Init(PsxInt32 xRes, PsxInt32 yRes,
                     PsxInt32 intl, PsxInt32 depth24);
Resolutions: 320x240
intl = GsNONINTER / GsINTER (0 or 1) + other stuff
depth24 = 0 for 16-bit, 1 for 24-bit



2. Buffer management

Use can use 2 display buffers to implement double-buffering.

void        GPU_DefDispBuff(PsxInt32 x0, PsxInt32 y0,
                            PsxInt32 x1, PsxInt32 y1);
x0, y0 = top left coord in VRAM of display buffer 0
x1, y1 = top left coord in VRAM of display buffer 1


PsxInt32    GPU_GetActiveBuff(void);

Returns the active buffer (0 or 1)


PsxInt32    GPU_VSync(void);

Wait for vertical blank.


void        GPU_EnableDisplay(PsxInt32 enable);

enable = 1 to enable the GPU to display stuff.


void        GPU_FlipDisplay(void);

Swaps the visible and working buffers.

8. Text output

Fonts are generated from a TIM file (see example fonts). Text output
can be scaled and shaded. Each text character drawn uses one primitive
in the primitive buffer (aka command buffer). See font.c for details.  

void InitFont(TIMHEADER *fontTIM, int dx, int dy, int n);
Prepares a font for use from a TIM. dx = width, dy = height of each
character in the TIM font. n = number of characters per line in the
TIM.
 
void SetCharSize(PsxUInt8 width, PsxUInt8 height);
This is used to set the output size of the font characters.


void SetCharLight(PsxUInt8 r, PsxUInt8 g, PsxUInt8 b);
This can be used to dim/fade a font.


void PrintChar(GsOT *ot, PsxUInt16 pos, PsxUInt16 xPos, PsxUInt16 yPos, PsxUInt16 chr);

void PrintString(GsOT *ot, PsxUInt16 pos, PsxUInt16 xPos, PsxUInt16 yPos, char *string);
Send a string of "text character" primitives to order table ot, at
order table depth pos. xPos & yPos are screen coordinates of starting
point for the string.
